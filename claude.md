Ik wil graag mijn vermogen groei inschatten de komende jaren.
Daarvoor heb ik verschillende gegevens

- Loon
- Bankrekeningen
- DeGiro portfolio
- NN werkgevers pensioen
- DeGiro persoonlijk pensioen
- Overwaarde woning

Ik wil graag een tool maken die op basis van huidige gegevens een voorspelling doet naar de toekomst.
Laten we de volgende Use cases nemen:
- Ik verwacht de komende jaren meer loon te verdienen en mijn kosten grotendeels gelijk te houden. Als we rekening houden met een bepaalde hoeveelheid inflatie wat zou dat 
betekenen voor mijn vermogen als ik 40 ben?
- De kans is enorm groot dat ik binnenkort mijn huis ga verkopen, hiermee komt een grote hoeveelheid overwaarde vrij. Welke vermogensopbouw tactieken heb ik tot mijn beschikking
rekening houdend met de Nederlandse wet- en regelgeving in Nederland?
- Als ik de vrijheid zou willen hebben om niet meer te "hoeven" werken wat zou mijn vermogen dan moeten zijn gebaseerd op verwachtte maandelijkse kosten?

Uiteindelijk wil ik een heel aantal webapps op elkaar laten aansluiten op dit thema dus ook deze app moet dat zijn. Ik wil het zelf nog wel makkelijke kunnen begrijpen, dus laten we de hele app, backend and frontend Javascript native maken. Met zoveel mogelijk volwassen libraries, React bijvoorbeeld. Maar ik zou graag dat je met betrekking tot technische vragen liever vraagt dan zelf kiest!
